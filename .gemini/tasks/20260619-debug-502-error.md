## 메모

- 원격 OCI 환경의 외부 대역폭 제한으로 인한 npx 다운로드 타임아웃 문제를 해결하기 위해, runner 스테이지에서 prisma를 글로벌 설치하여 완벽한 오프라인 구동 구조를 마련함.
- 이미 생성된 DB 스키마에 migrations 테이블이 유실되어 발생한 P3005 오류는 entrypoint 내에서 자동으로 감지해 `0_init`을 baseline 등록하고 재기동하게 만드는 자가 복구(Self-Healing) 로직을 탑재하여 정상화 완료함.
- 현재 Nginx, DB, App이 모두 healthy 상태이며 HTTPS 접속이 완벽히 복구됨.
