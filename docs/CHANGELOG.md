# 변경 이력 (CHANGELOG)

## [v0.2.0] - 2026-01-22

### 📱 모바일 UX 및 디자인 시스템 고도화

- **통합 디자인 시스템**: `p-3.5` 패딩, `2열 그리드 정보 배치`, `text-[10px]` 배지 크기 등 모바일 카드 표준 정립 및 전체 적용
- **초소형 필터 시스템**: 가로 스크롤 없이 한 화면에 노출되도록 필터 탭 레이블 간소화 및 크기 축소 (Ultra-Compact Tabs)
- **검색 기능 개선**: 모바일 접근성을 위해 명시적인 '검색' 버튼 추가
- **조직도 최적화**: 모바일 인덴트 축소 및 연결선 제거로 가로 공간 효율 극대화

### ⚡ 성능 및 인프라 최적화

- **PWA 진입 속도 향상**: 서비스 워커 `navigationPreload` 활성화 (로딩 속도 20~30% 개선)
- **화면 깜빡임(Flicker) 제거**: 클라이언트 리다이렉트를 서버 미들웨어로 이관
- **레이아웃 시프트(CLS) 방지**: 헤더 인증 영역 스켈레톤 최적화

### 🛠 관리자 및 보안 기능

- **시스템 설정 제어**: 미구현된 '일반 설정' 메뉴 임시 숨김 처리 (RBAC 유지)
- **접근성(A11y) 강화**: `DialogContent` 관련 ARIA 경고 해결 및 접근성 태그 보완

## [v0.1.0] - 2025-12-29

- 초기 릴리스 (Next.js 16 + React 19 + Prisma 6)
- 기본 SR 관리 기능 및 대시보드
- Docker 배포 환경 구성
