- **해결 필요**: NextAuth 설정 및 테스트 격리 개선

### 2. SR 생성 기능 미작동
- **현상**: E2E 테스트에서 SR 생성이 작동하지 않음

- **영향**: 1개 테스트 스킵
- **해결 필요**: 수동 테스트 및 디버깅

### 3. Select 컴포넌트 상호작용
- **현상**: shadcn/ui Select 컴포넌트 클릭이 복잡함

- **영향**: 테스트 안정성
- **해결 방법**: 더 나은 셀렉터 및 대기 시간 조정
